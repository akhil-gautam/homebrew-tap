# homebrew-tap

Homebrew formulas for [akhil-gautam](https://github.com/akhil-gautam) projects.

## Usage

```bash
brew tap akhil-gautam/tap
brew install <formula>
```

Available formulas:

- [`ecotoken`](https://github.com/akhil-gautam/ecotoken) — environmental impact dashboard for AI coding assistants.
- [`tokscale`](https://github.com/akhil-gautam/toktracker) — proactive insights for AI coding assistants (CLI).

Available casks:

- [`tokscale`](https://github.com/akhil-gautam/toktracker) — Tokscale menubar app + dashboard for macOS.

## Installing Tokscale

```bash
brew tap akhil-gautam/tap
brew install tokscale              # CLI (TUI)
brew install --cask tokscale       # Mac menubar app
```
